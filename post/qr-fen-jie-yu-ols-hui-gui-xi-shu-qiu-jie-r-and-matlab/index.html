matlab中如何用QR分解求回归系数

实际上，在matlb中求解β系数并不是用的hat matrix的形式，而是使用QR分解

malab中的regress函数中的核心代码如下：

clc ,clear
n=29;m=4;
q=load('./Ref/ho1.txt');
% size(q) 14行12列
x=[q(:,1:6);
    15 105.0 13.75 13.57 79.80 0.14;
    q(:,7:12);]
% 相当于认为地插入一行15 105.0 13.75 13.57 79.80 0.14
% 再把右侧的列拼接到左边下方来

y=[x(:,2)];
x1=[x(:,3)];
x2=[x(:,4)];
x3=[x(:,5)];
x4=[x(:,6)];
X=[ones(n,1),x1,x2,x3,x4]


%%%%%%%%%%% 以上为准备数据，下面3行即为regress函数的核心代码
[Q,R,perm] = qr(X,0)
b = zeros(5,1);
b(perm) = R \ (Q'*y);
% 等价于b(perm) = inv(R) * (Q'*y);

上述代码的数据文件:ho1.txt

得到的b(perm)即为beta系数

还是matlab中的直观，后续探索下如何把这种compat format分理处Q和R

Q = qr.Q(testmqr, complete = FALSE, Dvec =)
R = qr.R(testmqr, complete = FALSE)
solve(R) %*% t(Q)  %*% as.matrix(datain$y)
# 输出
          [,1]
ones 56.801136
x1   -2.775468
x2    6.256896
x4   11.505319

所以要想得到Q，就对qr函数的结果用qr.R函数，其他的以此类推
